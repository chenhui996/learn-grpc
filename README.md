# 需求

假设你要实现一个简单的 **任务管理应用**，其中包含以下功能：

1. 创建任务：
   1. 客户端 可以 向着 服务器 发送 任务请求 => 包括 任务名称 和 描述。
2. 获取任务列表：
   1. 客户端 可以 请求 服务器 返回 所有任务的列表。

> 现在，让我们按照以下步骤一步步实现这个示例：

  - **创建** 一个名为 **task.proto** 的文件.

    - **消息类型**
